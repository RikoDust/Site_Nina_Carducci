# Nina_Carducci


## 📋 À propos

Projet de formation consistant en l'optimisation, l'amélioration de l’accessibilité et le référencement du site existant d’une photographe.

Contexte : Projet de formation (Openclassrooms)

## ✨ Fonctionnalités principales

- Site web avec présentation d'activité
- Caroussel d'images
- Gallerie d'images
- Présentation de services
- Formulaire de contact
